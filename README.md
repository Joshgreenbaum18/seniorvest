# SeniorVest

**The premier platform for 1031 Exchange into Senior Living.**

Fractional TIC ownership in institutional-grade senior care facilities.

🌐 **Live:** https://joshgreenbaum18.github.io/seniorvest

## Stack
- React 18 + Babel (CDN) — zero build step
- Fonts: Cormorant Garamond + DM Sans
- Hosting: GitHub Pages

## Run locally
```bash
open index.html
```

## Connect to live pipeline
Replace SIGNALS, LISTINGS, HOLDINGS arrays with API calls to ~/v3_pipeline endpoints.
